🎵 Spotify Clone Web Application
A responsive Spotify Clone built using HTML, CSS, and JavaScript with features like music playback, progress bar, dynamic song updates, next/previous controls, and responsive design.
The application allows users to play, pause, switch, and control songs dynamically. Users can navigate between songs using next and previous buttons, view song durations, and interact with a real-time progress bar to seek audio playback. Song details such as song name, cover image, and duration are updated dynamically using JavaScript arrays and DOM manipulation.

The project also includes responsive design using CSS media queries, making the interface adaptable for both desktop and mobile devices. Font Awesome icons are used for music controls, and animated GIF effects enhance the UI experience.

✨ Features:

* Play/Pause music functionality
* Next and Previous song controls
* Dynamic song list using JavaScript
* Interactive progress/seek bar
* Song duration display
* Responsive design for mobile devices
* Dynamic song name and cover updates
* Clean Spotify-inspired user interface

🛠️ Technologies Used:

* HTML5
* CSS3
* JavaScript
* Font Awesome

This project helped strengthen understanding of:

* DOM Manipulation
* Event Listeners
* Arrays & Objects
* Audio API
* Responsive Web Design
* JavaScript Logic Building

A great beginner-friendly frontend project for practicing real-world JavaScript concepts and UI development.
